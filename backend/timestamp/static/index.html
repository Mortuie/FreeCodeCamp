Welcome to the Timestamp Microservice

Explanation:

This is the homepage of the application.

If you want to send a query to the api, please visit url/querystring.

After this the API will determine whether or not your querystring contains a valid date format, if it doesn't, you will get JSON will null as both entries. However if your querystring does contain a valid date. You will get a JSON back containing the unix and natural date.

Example:

url/December%2015,%202015

or

url/1450137600

will both yield:
JSON = { "unix": 1450137600, "natural": "December 15, 2015" }
